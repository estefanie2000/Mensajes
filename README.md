Descripción del proyecto

Este proyecto consiste en un script desarrollado en Python que automatiza el envío de mensajes personalizados y archivos adjuntos a través de WhatsApp Web, utilizando información almacenada en un archivo Excel.

El objetivo principal es optimizar procesos repetitivos que requieren enviar mensajes individuales a múltiples contactos, reduciendo el tiempo de ejecución y minimizando errores humanos.

El script lee los datos desde Excel, genera mensajes personalizados con el nombre del destinatario y envía automáticamente el mensaje junto con un archivo asociado a cada registro.

Funcionalidades principales

Lectura de datos desde un archivo Excel.

Personalización de mensajes con información del contacto.

Envío automático de mensajes mediante WhatsApp Web.

Envío de archivos adjuntos asociados a cada contacto.

Manejo de errores y validación de datos.

Control de tiempos entre envíos para evitar bloqueos.
